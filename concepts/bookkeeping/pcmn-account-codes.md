# PCMN/MAR account codes (sub-account level)

Sub-account detail within the legally mandated chart ([[chart-of-accounts-mar]], RD 29-04-2019). Not exhaustive , the slice a one-person management BV actually posts to. The firm's real chart (imported from the books) is authoritative.

## Equity, provisions, long-term debt (class 1)

| Code | Account |
|---|---|
| 10 | Capital |
| 13 / 130 / 133 | Reserves / legal reserve / available reserves |
| 14 | Retained earnings (or accumulated loss) |
| 16 | Provisions for liabilities and charges |
| 17 | Amounts payable after more than one year |

## Fixed assets (class 2)

| Code | Account |
|---|---|
| 20 | Formation costs ([[formation-costs-advies]]) |
| 21 / 22-27 | Intangible / tangible fixed assets (24 = furniture & vehicles) |
| ...9 | Accumulated depreciation (contra) ([[depreciation]]) |

## Receivables, cash, payables (classes 4-5)

| Code | Account |
|---|---|
| 40 / 400 | Trade debtors |
| 411 | **VAT recoverable** |
| 416 / 48 | Other receivables / miscellaneous |
| 44 / 440 | Trade creditors |
| 451 | **VAT payable** |
| 453 | Withholding tax payable (bedrijfsvoorheffing, see [[withholding-on-director-pay]]) |
| 454-459 | Social security / remuneration payable |
| 489 | Director current account ([[director-loan-rekening-courant]]) |
| 490-493 | Accruals / deferrals (cut-off) |
| 550 / 570 | Bank / cash |
